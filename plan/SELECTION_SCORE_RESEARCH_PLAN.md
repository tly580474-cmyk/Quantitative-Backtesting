# 短线评分与波段评分研究计划

## 0. 当前决策

现有 `selectionScore` 不再作为新评分体系的默认设计基础，也不通过简单调整原有七个维度的权重继续优化。
后续研究拆分为三层：

1. **短线评分**：评价未来 1–5 个交易日的可交易机会；
2. **波段评分**：评价未来 20–40 个交易日的风险调整后机会；
3. **风险与可交易性门槛**：独立于收益评分，负责 ST、停牌、流动性、涨跌停和数据质量过滤。

两种评分允许给出不同甚至相反的结果。例如“波段 85、短线 42”表示中期结构较好，但当前买点不佳。
风险门槛未通过时显示“不参与评分/不可交易”，不再统一压入 0 分档。

本计划只定义研究、验证和验收流程。在研究门槛通过前，不替换前端现有评分，也不把候选规则解释为实盘买入信号。

## 1. 研究背景与基线结论

现有评分回测区间为 2020-01-01 至 2026-07-13，覆盖 5,713 只股票和 7,365,107 个评分日。
基于 `selection_score_fine.csv` 的 5 分档结果，分数与各持有期平均收益的 Spearman 相关大致为：

| 持有期 | Spearman 相关 | 初步解释 |
| --- | ---: | --- |
| 1 日 | +0.31 | 存在弱短期延续信息 |
| 3 日 | +0.34 | 存在弱短期延续信息 |
| 5 日 | +0.28 | 存在弱短期延续信息 |
| 20 日 | -0.84 | 明显转为反向关系 |
| 50 日 | -0.82 | 高分没有形成波段优势 |
| 100 日 | -0.89 | 长周期反向关系明显 |
| 120 日 | -0.70 | 长周期仍无正向单调性 |

因此，现有总分把短期动量、突破、均线趋势、支撑位置、低波动和风险扣分混合后，无法同时代表短线和波段机会。
本研究的第一原则是**先固定预测期限和交易口径，再研究因子与评分**。

### 1.1 现有回测必须修正的偏差

1. 当前按评分日 T 收盘成交，实盘无法在看到完整收盘数据后仍以 T 日收盘价买入；
2. 每只股票每天产生一条未来收益，20–120 日标签高度重叠，不能把全部记录视为独立样本；
3. 股票数据提前结束时缺少未来价格，退市、长期停牌等极端损失可能被遗漏；
4. `is_st` 虽被读取，但当前评分函数没有实际使用；
5. 前端使用成交量估算成交额，研究脚本使用数据库真实成交额，口径不一致；
6. 原回测未完整处理次日涨停不可买、跌停不可卖、滑点、手续费和印花税；
7. 原始收益没有充分剥离市场、行业、市值和流动性暴露；
8. 0–5 分档混有强制低流动性归零样本，不能直接解释为可交易的反转组合。

## 2. 研究目标与非目标

### 2.1 研究目标

1. 建立严格错期、可复现、可交易的评分研究数据集；
2. 分别验证短线和波段候选因子的样本外预测能力；
3. 建立行业、市值和流动性中性化后的横截面评分；
4. 使分数档位与目标期费后超额收益尽量保持单调；
5. 给出评分置信度、覆盖率、换手、容量和失效环境；
6. 形成可以接入自选股排序和全市场选股的版本化评分协议。

### 2.2 非目标

- 不根据本次全样本结果直接拍定人工权重；
- 不把低分反转简单等同于“买入 ST 或流动性枯竭股票”；
- 不以提高历史累计收益作为唯一优化目标；
- 不在本阶段研究分钟级高频、自动下单或组合仓位管理；
- 不承诺两个评分在所有年份和市场状态下持续有效。

## 3. 核心研究假设

| 编号 | 假设 | 验证方式 |
| --- | --- | --- |
| H1 | 原评分的有效期限主要集中在 1–5 日 | 比较原分数在不同未来期限的 Rank IC 与分层收益 |
| H2 | 20 日以上的反向结果部分来自短期过热和风险样本偏差 | 分别控制 ST、停牌、退市、流动性、行业、市值和市场收益 |
| H3 | 短线延续与短线反转依赖市场状态 | 按指数趋势、波动率、成交额和行业强弱分组验证交互效应 |
| H4 | 波段预测应依靠慢变量，短期 K 线指标只适合决定入场时机 | 对慢变量、快变量及二者组合进行消融实验 |
| H5 | 独立风险门槛优于把风险事件折算成负分 | 比较硬过滤与风险扣分对尾部损失、覆盖率和收益单调性的影响 |
| H6 | 横截面排序和中性化优于固定技术阈值累加 | 比较固定规则分、截面 Rank 分和滚动模型分 |

研究结果允许推翻上述假设。未通过样本外验证的假设不得进入产品评分。

## 4. 评分定义

### 4.1 短线评分

**用途**：判断当前是否存在 1–5 个交易日的可交易机会和入场时机。

**主标签**：

```text
T 日收盘生成特征
T+1 日开盘按可成交规则买入
T+3 / T+5 日收盘退出
标签 = 个股费后收益 - 同期基准或行业收益
```

3 日和 5 日分别评价，不先强行合成；只有两者方向和稳定性一致时才形成短线总分。

**更新频率**：每个交易日收盘后。

**候选因子组**：

- 1/3/5 日收益、隔夜跳空、收盘位置和短期残差动量；
- 超跌幅度、快速修复、下影承接和短期反转；
- 突破距离、突破后保持程度、量价确认；
- 上涨日/下跌日成交量差异、量比、换手变化和成交额；
- 相对行业、相对指数的短期强弱；
- ATR、振幅、冲高回落、连续大阴线和尾部波动；
- 市场趋势、行业状态与“延续/反转”因子的交互项；
- 涨停不可买、开盘跳空过高等执行风险。

### 4.2 波段评分

**用途**：评价未来约一个月到两个月的中期结构与持续性，不负责精确买点。

**主标签**：

```text
T 日收盘生成特征
T+1 日开盘按可成交规则买入
T+20 日收盘退出（主标签）
T+40 日收盘退出（稳定性辅助标签）
标签 = 个股费后收益 - 行业与市场中性收益
```

**更新频率**：可以每日计算，但产品提示应强调其变化较慢；后续可评估改为每周刷新。

**候选因子组**：

- MA20/60/120 的位置、斜率、排列和趋势稳定性；
- 20/60/120 日相对指数、相对行业的残差强弱；
- 中期反转、距离阶段高低点、回撤修复速度；
- OBV、量价累积、换手稳定性和资金持续性代理变量；
- 20/60 日波动率、最大回撤、下行波动与收益/风险比；
- 行业趋势、市场宽度和指数状态；
- 上市时间、规模、流动性和历史停牌等控制变量。

MACD、RSI、单日 K 线和 1–3 日量比不得在波段评分中占据主要权重。它们可以作为入场辅助，但必须通过增量贡献检验。

### 4.3 风险与可交易性门槛

以下规则独立于两个收益评分：

- ST、退市整理或明确退市状态；
- 停牌、价格或成交量数据无效；
- 上市天数不足研究门槛；
- 20 日成交额、有效交易天数或成交连续性不足；
- T+1 一字涨停无法买入；
- 极端连续跌停或退出日无法卖出；
- 复权、公司行动或股票主数据无法可靠时点化；
- 缺失关键行业、市值或基准数据。

风险门槛输出 `eligible / caution / blocked / unknown`。`blocked` 和 `unknown` 不生成收益评分。

## 5. 数据集与时点规则

### 5.1 股票池

首轮使用全 A 股普通股票，并保留板块标记。每个交易日按当时可知信息构造股票池，避免用当前股票列表回看历史。

必须支持以下切片：

- 沪深主板、创业板、科创板、北交所；
- 大/中/小市值；
- 行业；
- 高/中/低流动性；
- 上市时间分组；
- ST 与非 ST；
- 牛市、熊市、震荡市及高低波动状态。

### 5.2 数据要求

- 前复权或后复权口径在特征和收益之间保持一致；
- 行情、ST、行业、市值、上市/退市、停牌和涨跌停状态均采用时点化数据；
- 数据快照、查询区间、代码版本和配置必须写入实验 manifest；
- 对退市、长期停牌和退出受限样本定义明确的收益处理规则，不允许静默丢弃；
- 前端与研究侧统一成交额、成交量单位及量比口径。

### 5.3 成交假设

基线口径：

- 信号在 T 日收盘后生成；
- 最早 T+1 开盘成交；
- 买卖均计滑点和佣金，卖出计印花税；
- T+1 不可买时记录为未成交，不允许用后续价格自动补成交，除非该规则单独建模；
- 退出日不可卖时顺延至首个可卖日，并单独统计延迟风险；
- 同时报告毛收益、费后收益和相对收益。

## 6. 研究方法

### 6.1 单因子评价

每个候选因子至少报告：

- 覆盖率、缺失率、分布和异常值；
- Pearson IC、Spearman Rank IC、ICIR 和方向胜率；
- 十分位或五分位收益及其单调性；
- Top-Bottom 组合毛收益、费后收益、换手和最大回撤；
- 1/3/5/10/20/40 日衰减曲线；
- 年度、板块、行业、市值、流动性和市场状态稳定性；
- 与已有因子的相关性和新增解释力。

高度重叠的未来收益使用日期聚类、股票聚类或双向聚类稳健标准误；组合回测同时采用按日横截面建仓，避免把重叠观测数误当作独立样本数。

### 6.2 预处理

统一比较以下口径：

1. 原始值；
2. MAD 或分位数去极值；
3. 横截面 Rank 或 z-score 标准化；
4. 行业中性化；
5. 行业 + 对数市值 + 流动性中性化。

只有在中性化后仍有稳定预测力的因子，才优先进入评分模型。

### 6.3 评分模型候选

按由简单到复杂的顺序比较：

1. 等权 Rank 合成；
2. 滚动 Rank IC / ICIR 加权；
3. 带正则化的线性或有序模型；
4. 能表达市场状态交互的树模型。

固定阈值加分法作为对照组保留。复杂模型只有在最终盲测集、成本后指标和稳定性上显著优于简单模型时才采用。

### 6.4 分数校准

最终 0–100 分优先定义为当日可交易股票池的预测值百分位：

- 50：当日股票池中位水平；
- 80：位于预测值前 20%；
- 90：位于预测值前 10%。

同时保存原始预测值、横截面排名、模型版本和置信度。评分阈值不再直接沿用原来的 75/60/45 分层。

## 7. 样本外验证设计

### 7.1 时间切分

建议第一轮固定为：

- 训练集：2020-01-01 至 2023-12-31；
- 验证集：2024-01-01 至 2025-06-30；
- 最终盲测集：2025-07-01 至数据快照截止日。

具体日期可根据数据完整性调整，但最终盲测集在模型和阈值冻结前不得用于选因子或调权重。

在固定切分之外，再进行滚动 walk-forward：仅使用过去数据确定因子方向、权重和校准参数，然后评价下一段时间。

### 7.2 防止过拟合

- 登记每次实验的因子、参数、样本区间和结果，不只保存最好结果；
- 控制候选特征和参数搜索空间；
- 对多重检验进行修正，并报告候选数量；
- 做因子消融、参数扰动和成本压力测试；
- 不因某一年或某种市场状态表现突出就直接上线；
- 最终盲测失败后，不允许继续反复查看同一盲测集调参并仍称其为盲测。

## 8. 分阶段实施

### 阶段 A：审计与基线重跑（P0）

**任务**：

1. 对齐前端和 Python 评分实现；
2. 修复 ST 未使用、成交额口径不一和退市/停牌样本遗漏；
3. 将成交改为 T+1 开盘并加入完整交易约束；
4. 输出原评分的 IC、Rank IC、年度分层、市场状态分层及成本后结果；
5. 建立固定数据快照和实验 manifest。

**退出门槛**：相同版本和配置可以重复生成一致结果；缺失未来收益的原因均可审计。

### 阶段 B：候选因子库与单因子研究（P0）

**任务**：

1. 按短线、波段、风险控制分类登记候选因子；
2. 生成多持有期单因子报告；
3. 完成行业/市值/流动性中性化比较；
4. 建立因子相关性、衰减和市场状态分析；
5. 删除无增量、方向不稳或成本后失效的因子。

**退出门槛**：形成两份候选因子白名单和一份淘汰清单，每个结论可追溯到实验结果。

### 阶段 C：双评分原型（P1）

**任务**：

1. 分别训练短线与波段简单基线模型；
2. 比较等权、IC 加权、正则线性模型和状态交互模型；
3. 进行消融实验，确认每个因子的边际贡献；
4. 生成 0–100 横截面百分位及置信度；
5. 比较原评分、短线新评分和波段新评分。

**退出门槛**：验证集上两个评分均满足各自的最低验收标准，且不存在明显的数据泄漏或不可成交收益。

### 阶段 D：盲测与压力测试（P1）

**任务**：

1. 冻结因子、参数、权重、过滤和校准规则；
2. 一次性运行最终盲测；
3. 进行 1 倍、2 倍滑点和容量压力测试；
4. 检查行业集中度、小盘暴露和极端行情表现；
5. 记录失败条件和适用边界。

**退出门槛**：盲测通过后才允许设计产品替换方案；未通过则回到新一轮研究并建立新的未来验证窗口。

### 阶段 E：产品接入与观察期（P2）

**任务**：

1. 接口同时返回短线分、波段分、风险状态、置信度和版本；
2. 自选股支持按任一评分排序，不再默认只有一个总分；
3. 详情页解释主要贡献项，但不把相关性描述为确定性因果；
4. 旧评分保留为历史对照，不继续作为“核心优选池”依据；
5. 先进行不少于 20 个交易日的影子观察，再决定是否默认启用。

## 9. 最低验收标准

短线和波段分别验收，不允许一个评分的成绩弥补另一个评分失败。

### 9.1 共同标准

- 最终盲测方向与训练/验证一致；
- 评分前 20% 的费后超额收益高于后 20%；
- 主要分档收益总体单调，不依赖单个极端分档；
- 行业、市值和流动性中性化后仍保留有效性；
- 不依赖 ST、不可成交或将退市股票贡献主要收益；
- 至少多数年度和主要市场状态方向一致；
- 结果包含置信区间或稳健显著性评价，而非只展示样本均值；
- 2 倍基线滑点下仍不出现结论性反转。

### 9.2 短线专项标准

- 3 日和 5 日 Rank IC 方向一致；
- 成本后 Top-Bottom 差保持为正；
- 换手、涨停不可买率和开盘跳空侵蚀可接受；
- 延续与反转模式的适用状态有明确说明。

### 9.3 波段专项标准

- 20 日主标签与 40 日辅助标签方向一致或具有可解释差异；
- 结果不主要来自市场 Beta、单一行业或微盘股暴露；
- 费后超额收益、最大回撤和收益单调性均优于原评分；
- 快变量删除后核心效果不应完全消失。

## 10. 研究交付物

1. `selection_score_audit_report`：原评分审计与修正后基线；
2. `short_score_factor_report`：短线单因子、衰减和状态分析；
3. `swing_score_factor_report`：波段单因子、中性化和稳定性分析；
4. `score_experiment_registry`：所有实验配置、数据版本和结果索引；
5. `short_score_spec` 与 `swing_score_spec`：因子、方向、模型、校准和版本协议；
6. 双评分样本外/盲测报告；
7. 成本、容量、停牌、涨跌停和退市压力测试报告；
8. 产品接口与界面改造方案；
9. 旧评分下线或降级决策记录。

## 11. 停止条件

出现以下任一情况，应停止上线推进并回到研究阶段：

- 有效性只存在于全样本，样本外消失；
- 收益主要由不可交易股票或极少数异常样本贡献；
- 中性化或加入真实成本后方向反转；
- 评分档位不单调且无法稳定复现；
- 权重对训练区间或参数细微变化极度敏感；
- 两个评分高度相关，实际没有形成不同持有期的信息分工；
- 数据时点、复权、退市或成交约束仍无法可靠处理。

```text
        ↓
时点数据与可交易性检查
        ├── blocked / unknown → 不评分，展示原因
        ↓ eligible / caution
短线模型（1–5 日目标）     波段模型（20–40 日目标）
        ↓                         ↓
短线 0–100 + 置信度        波段 0–100 + 置信度
        └──────────┬──────────────┘
                   ↓
风险状态、主要贡献项、适用期限和模型版本
```

最终产品不再回答“这只股票总体有多少分”，而是分别回答：

1. **现在是不是一个较好的短线入场机会？**
2. **未来一个月至两个月是否具备较好的波段结构？**
3. **即使预期收益较高，它当前是否真实可交易、风险是否可接受？**
