# 选股评分打分规则

本文档说明前端「选股评分」模块的打分规则。规则定义见源码 `src/features/marketData/selectionScore.ts`，是一套**规则型正向打分 + 风控倒扣**的宽松初筛体系，与策略回测用的 `multiFactor.ts` Z-score 量化合成体系相互独立（区别见文末）。

## 1. 用途与定位

- 用于股票池的**技术面宽松初筛**，输出 0–100 分，并归入四个分层池。
- **不是买入信号**。文档明确标注：本模型用于宽松初筛，冷却池建议 20–40 个交易日后重新评分。
- 相对强弱项需要传入沪深 300 日 K（`benchmarkCandles`）；若数据不足，相关加分项直接不生效。

## 2. 计算前置条件

- 至少需要 **65 根有效日 K**（需计算 60 日均线趋势与完整评分）。
- 不足时返回 `status: 'insufficient'`，提示当前根数与最低要求。
- 有效 K 线判定：`open / close / high / low / volume` 均须为有限数值，并按日期升序排序。

## 3. 打分总机制

评分由两部分组成：

1. **正向得分（满分 100）**：7 个维度各自按"命中加分 / 不命中不扣分"的固定阈值计分。每节得分在 0 至该节满分之间截断：

   ```
   sectionScore = min(满分, max(0, 本节各项之和))
   ```

   七节求和得 `rawPositiveScore`（范围 0–100）。

2. **风控倒扣分（无上限）**：单独一节 `risk`，命中即直接扣分（可为负），不在 0–100 内截断。

**最终分数计算：**

```
normalizedBaseScore = round(rawPositiveScore / 100 × 100)   // 即 0–100
riskDeduction       = |风控各项之和|
score              = clamp(0, 100, normalizedBaseScore − riskDeduction)
```

## 4. 七个正向维度

### 4.1 中长期均线趋势（满分 22）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 60 日均线向上或走平 | +8 | MA60 在 5 个交易日前后变化 ≥ −0.5% |
| 20 日线在 60 日线上方或即将上穿 | +5 | MA20 ≥ MA60 × 0.98 |
| 股价站上或距离 20 日线不超过 5% | +5 | 收盘价 ≥ MA20，或距 MA20 ≤ 5% |
| 5 日、10 日线至少一条向上 | +4 | MA5 ≥ 前 5 日 MA5，或 MA10 ≥ 前 5 日 MA10 |
| 60 日均线明显向下 | −6 | MA60 前后变化 < −1.5% |
| 跌破 60 日线且 3 日未收回 | −6 | 最近 3 日收盘均低于 MA60 |

### 4.2 短期动量强弱（满分 18）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 近 20 日表现强于沪深 300 | +6 | 个股 20 日收益 > 沪深 300 20 日收益（需基准数据） |
| 近 10 日涨幅为正 | +5 | 10 日收益率 > 0 |
| 近 5 日无连续 3 天明显下跌 | +4 | 无连续 3 日单日跌幅 ≥ 2% |
| 距离 20 日阶段高点不超过 5% | +3 | 距前 20 日最高价 ≤ 5% |
| 近 20 日跌幅超过 18% | −6 | 20 日收益率 < −18% |
| 近 5 日出现连续放量下跌 | −5 | 最近 3 日连续下跌且成交量不低于 20 日均量 |

### 4.3 成交量资金验证（满分 18）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 上涨或突破日量能达到 1.2 倍 | +6 | 当日上涨或收盘 ≥ 前 20 日高点 98%，且量比 ≥ 1.2 |
| 上涨日量能高于下跌日 | +5 | 近 10 日上涨日 ≥ 2、下跌日 ≥ 1，且上涨日均量 > 下跌日均量 |
| 回调时量能明显缩小 | +4 | 下跌日均量 < 上涨日均量 × 0.9 |
| 近 3 日均量不低于 20 日均量 80% | +3 | 3 日均量 ≥ 20 日均量 × 0.8 |
| 持续缩量阴跌 | −5 | 近 5 日收益 < 0 且 5 日均量 < 20 日均量 × 0.6 |
| 放量下跌且无明显承接 | −6 | 当日跌 ≤ −3%、量比 ≥ 1.5，且（收盘−最低）/（最高−最低）< 0.25 |

### 4.4 支撑压力位置（满分 14）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 距离 20 日线或支撑位不超过 8% | +5 | 距 MA20 ≤ 8% |
| 距离上方压力仍有 5% 以上空间 | +4 | 距前 60 日最高价 ≥ 5% |
| 回踩支撑位未有效跌破 | +3 | 最近 5 日低点均 ≥ MA20 × 0.95 |
| 小阳线、十字星或缩量止跌 | +2 | 距支撑 ≤ 8% 且（收阳 或 近似十字星 或 缩量） |
| 距离上方压力不足 3% | −4 | 距前 60 日最高价 < 3% |
| 明显远离 20 日线超过 12% | −5 | 距 MA20 > 12% |

### 4.5 K 线形态结构（满分 10）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 箱体平台尝试突破 | +4 | 收盘价 ≥ 前 20 日高点 × 0.97 |
| 突破平台后未明显跌回 | +4 | 近 5 日最高 ≥ 前 21–25 日平台高点，且现价 ≥ 平台 × 0.98 |
| 底部逐步抬高 | +4 | 近 15–10 日低点 < 近 10–5 日低点 < 近 5 日低点 |
| 上升三角形 / 收敛 / 旗形整理 | +4 | 近 5 日振幅均值 < 前 10–15 日振幅均值 × 0.8，且现价 ≥ MA20 |
| 连续小阳线或温和上行 | +3 | 近 5 日收益 > 0、阳线 ≥ 3 天、每日涨跌幅绝对值 < 5% |

### 4.6 MACD / RSI 辅助指标（满分 8）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| MACD 金叉或即将金叉 | +3 | DIF ≥ DEA，或差值改善且绝对值极小 |
| 红柱扩大或绿柱缩短 | +2 | 当前 MACD 柱 > 前一根柱 |
| RSI14 位于 35–75 | +3 | 35 ≤ RSI14 ≤ 75 |
| RSI14 高于 85 | −3 | RSI14 > 85 |
| MACD 死叉且绿柱持续扩大 | −3 | DIF<DEA、柱为负且持续扩大 |

### 4.7 波动与回撤控制（满分 10）

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 近 20 日最大回撤小于 12% | +4 | 滚动最大回撤 < 12% |
| 近 10 日无连续大阴线 | +3 | 无连续 2 根实体跌幅 ≥ 3% 的阴线 |
| 波动平稳、冲高回落不频繁 | +3 | 近 20 日明显冲高回落 ≤ 2 次 |
| 近 20 日最大回撤超过 20% | −5 | 滚动最大回撤 > 20% |
| 连续大阴线后未企稳 | −6 | 连续大阴线且最近 5 日未站稳 MA5 |

## 5. 风控倒扣项（无上限，直接减分）

风控节不截断下限，命中即扣分，最终可能把总分拉到 0。

| 规则 | 分值 | 命中条件 |
| --- | --- | --- |
| 高位长上影放量回落 | −6 | 距前高 < 8%、收阴、上影 > 实体 2 倍且量比 ≥ 1.3 |
| 连续放量大跌 | −8 | 最近 2 日均跌 > 2.5% 且放量（量 ≥ 20 日均量） |
| 跌破 20 日线和 60 日线且未收回 | −8 | 最近 3 日收盘均低于 MA20 与 MA60 |
| 均线明显空头排列 | −10 | MA5<MA10<MA20<MA60 且三条均线均下行 |
| 短期连续大阴线且无企稳 | −6 | 近 10 日连续大阴线且最近 5 日未企稳 |
| 日均成交额低于 3000 万 | −5 | 估算 20 日均成交额 < 3000 万 |
| 日均成交额低于 1000 万（强制冷却） | 触发 blocked | 估算 20 日均成交额 < 1000 万，无视分数直接进冷却池 |

> 流动性估算：历史 K 线未提供成交额，代码用 `成交量 × 典型价 × 100 股/手` 估算（典型价取 (高+低+收)/3），结果约以亿元计。

## 6. 综合分与分层

```
score = clamp(0, 100, normalizedBaseScore − |风控倒扣|)
```

分层阈值（`forcedCooling` 时强制为 blocked）：

| 分数 | 层级 | 标签 | 含义 |
| --- | --- | --- | --- |
| ≥ 75 | core | 核心优选池 | 趋势较好、量价健康，重点跟踪，等回踩或突破确认 |
| ≥ 60 | watch | 观察备选池 | 结构不差，等补量、突破或回踩企稳 |
| ≥ 45 | weak | 弱势观察池 | 信号尚未统一，暂不操作，保留观察等趋势修复 |
| < 45 或日均额 < 1000 万 | blocked | 冷却剔除池 | 短期结构偏弱，移出选股池，20–40 交易日后重评 |

## 7. 重要边界与注意事项

- **非买入信号**：宽松初筛，冷却池建议 20–40 交易日后重新评分。
- **流动性估算依赖成交量**：历史 K 无成交额字段，用 `成交量 × 典型价 × 100` 估算，与真实成交额存在偏差。
- **无行业 / 市值中性化**：目前仅做相对沪深 300 的强弱对比；行业中性化在 `plan/PHASE5_PLAN.md` 规划中，尚未实现。
- **固定阈值可复现**：诸如"即将上穿、重要支撑、形态偏强、明显下跌"等描述均量化为固定阈值，便于复现。

## 8. 与多因子合成体系的区别

项目内另有一套独立的 `src/features/strategies/multiFactor.ts` 评分体系，用途不同，勿混淆：

| 维度 | 选股评分（selectionScore.ts） | 多因子合成（multiFactor.ts） |
| --- | --- | --- |
| 方法 | 规则型固定阈值加分/扣分 | 滚动窗口 Z-score 标准化加权 |
| 输出 | 0–100 分 + 四级分层 | 合成因子值，用于买卖信号 |
| 用途 | 股票池技术面宽松初筛 | 策略回测信号生成 |
| 中性化 | 无（仅相对沪深 300） | 文档支持等权 / IC / RankIC / 手动权重与行业中性化 |

多因子合成中，波动率组（vol_5=0.5、vol_10=0.15、vol_20=0.35）与反转组（多个反转因子各取负权重）分别标准化后加权求和，再由 `compositeFactorStrategy.ts` 等权相加得到买卖信号；详见 `doc/FACTOR_RESEARCH_USER_GUIDE.md`。
